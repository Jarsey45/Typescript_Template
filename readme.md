# Starter Project 
## Typescript

## Install

```
$ git clone https://github.com/Jarsey45/Typescript_Template directory_name
```

## Start

```
npm start 
```
#### or typescript watch
``` 
npm run tsc-watch
```
### Comments
If you want to import other .ts file just add this line.

```
///<reference path="Anal.ts" />
```
